# How to Deploy to GitHub Pages

Follow these steps to host your Railway Analytics Hub:

1. **Create a NEW Repository**:
   - Go to [GitHub](https://github.com) and create a new repository (e.g., `railway-analytics`).
   - Set it to **Public**.

2. **Upload the Files**:
   - Inside your repository, click **"Add file"** -> **"Upload files"**.
   - Drag and drop ALL files from the `gh-pages` folder:
     - `index.html`
     - `intermodal_chart.html`
     - `professional_chart.html`
     - `intermodal_data.js`
     - `embedded_data.js`
   - Click **"Commit changes"**.

3. **Enable GitHub Pages**:
   - Go to the **"Settings"** tab of your repository.
   - Click on **"Pages"** in the left sidebar.
   - Under **"Build and deployment"** > **"Branch"**, select `main` (or `master`) and Click **"Save"**.

4. **Share on LinkedIn**:
   - Wait about 1-2 minutes.
   - GitHub will give you a URL (e.g., `https://username.github.io/railway-analytics/`).
   - You can now use this link in your LinkedIn posts!

*Note: All data is embedded in the JS files, so no backend is needed.*
